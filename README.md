# NEMS Feeds Site

## How to run locally

1. create a .env file and save to the root directory and enter details for the following

```sh
NEMS_SEMP_URL=https://broker:943
NEMS_SEMP_USER=username
NEMS_SEMP_PASSWORD=password
NEMS_VPN=vpn
```

1. `npm install`
1. `npm start`

## Node Requirements

Node 18
